ConfbridgeTalking

Synopsis

Raised when a conference participant has started or stopped talking.

Syntax

Event: ConfbridgeTalking
Conference: <value>
TalkingStatus: <value>
Channel: <value>
Uniqueid: <value>
Conference: <value>
Arguments
  • Conference - The name of the Confbridge conference.
  • TalkingStatus
    • on
    • off
  • Channel
  • Uniqueid
  • Conference
