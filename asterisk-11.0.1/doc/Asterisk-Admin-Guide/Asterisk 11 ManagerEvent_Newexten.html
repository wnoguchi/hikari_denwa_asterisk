Newexten

Synopsis

Raised when a channel enters a new context, extension, priority.

Syntax

Event: Newexten
Application: <value>
AppData: <value>
Channel: <value>
Context: <value>
Extension: <value>
Priority: <value>
Uniqueid: <value>
Arguments
  • Application - The application about to be executed.
  • AppData - The data to be passed to the application.
  • Channel
  • Context
  • Extension
  • Priority
  • Uniqueid
